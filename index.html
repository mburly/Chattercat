What is Chattercat? Chattercat is an application powered by Python that listens to Twitch.tv IRC chatrooms and tracks emote frequency in messages.

Emotes, similar to emojis, are icons used as reactions that appear in place of text in Twitch messages. Twitch provides a default set of emotes, while third-party browser extensions such as BetterTTV and FrankerFaceZ allow channels to add custom emotes to their chatroom.
